# React-Webpack_Starter
Create React Without Create-React-App CLI

``` bash
# Install Dependencies
npm install

# Initialize & Add Package.json
npm init -y

# Install Production Dependencies
npm i react react-dom
or
yarn install
yarn install react react add

# Install Webpack Dev Dependencies
npm i --save-dev webpack webpack-dev-server webpack-cli

#Install Babel
npm i --save-dev babel-core babel-loader babel-preset-env babel-preset-react html-webpack-plugin

# Serve on localhost:3000
npm start

# Build for Production
npm run build
```
